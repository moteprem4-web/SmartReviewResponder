#Review Analyzer & Response

Interactive Review Analysis Application using LangGraph and Gemini API

Overview

This project is an interactive review analysis tool that allows users to input reviews and receive automated, context-aware responses. The system leverages the LangGraph workflow for structured processing and the Gemini API for intelligent response generation.

Features

Accepts user reviews via a simple interface.

Analyzes the sentiment, intent, and context of reviews.

Generates automated, relevant responses to each review.

Built using LangGraph for workflow management.

Utilizes Gemini API for AI-driven response generation.
